Introduction

This is the beginning of a proposed Application programming interface for C code. The primary purpose is an interface that can be called by python using ctypes but it could also be used by other code.

The following header files are discussed:

pvDataPy.h
C++ suport that creates a wrapper for each of the basic pvData types.
pvDataPyCAPI.h
This is a set of C functions that make the pvDataPY wrappers available to C code. The address of a wrapper is passed between C code and C++ code.
pvDataPyCreateTest.h
An example of code that creates a top level PVStructure and a C callable function that creates a top level "wrapper" that C code calls. There is also a C callable function to delete the top level wrapper. Calling this function also deletes the top level PVStructure.

pvDataPy.h

This is a complete implementation of the wrappers. They are desigend to be usable by C++ code. In addition to the use by pvDataPyCAPI.h they can also be used by different pieces of C++ code that are called be Python Python can pass addresses of wrappers between the different pieces.

pvDataPyCAPI.h
